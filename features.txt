Документация

Для целостности данных пара объект-операция должна быть уникальна в правилах. 
Период для правил задается в секундах.

Feature list

- 1 правило, возвращающее ДА если количество запросов превышает указанное число в правиле +
- правило поддерживает две операции "больше чем" и "равно" +
- правило поддерживает указание объекта для которого оно будет выполняться +
- подсчитывать кол-во проверок для переданного объекта +
- возможность задать несколько правил для одного объекта +
- возможность работы с несколькими объектами +
- добавить возможность работы правил с периодом (только в секундах). Начало периода, считается стартом программы +
- Проверка периода между запросами (1 правило - 1 объект) +
- Проверка периода между запросами и правилами (каждый объект может иметь несколько правил) +

- если период не указан, или период 0 то работаем без периода
